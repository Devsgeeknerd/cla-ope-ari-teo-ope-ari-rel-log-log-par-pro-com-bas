# Operadores Aritméticos — Teoria

***Conteúdo da Aula:***

Os operadores aritméticos são usados para desenvolver operações matemáticas:

| Operador | Descrição |
| :------: | :-------: |
| `+` | Operador de adição |
| `-` | Operador de subtração |
| `*` | Operador de multiplicação |
| `/` | Operador de divisão |
| `%` | Operador de módulo (resto da divisão) |

Por exemplo:

```javascript
numero1 : inteiro = 1;
numero2 : inteiro = 1;
resultado : inteiro = numero1 + numero2;
```

No exemplo acima, a variável `resultado` receberá o valor 2, que é o resultado da soma do conteúdo das variáveis `numero1` e `numero2`.
